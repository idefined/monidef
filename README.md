==== 程序说明 ====

1.文件结构
------------------------------------------------------------------
 - moni.sh
  - SHELL脚本程序，cron上做任务计划，一个小时执行一次
 
 - monitor.conf
  - 需要监控的主机以端口的配置文件
  - 填写格式"ip:port|port"
  - run.log
  - 脚本运行时产生的日志文件
  - "#"是注释符

 - mail.conf
  - 收件人列表
  - "#"是注释符
2.参数传入
 - 例子： moni.sh 192.168.1.1:80\|8080 如果是多个端口需要用"\"转义符
 - 只检测传入的IP以及端口不做日志记录
3.ping 以及 nc 命令的参数可在脚本里面调整
